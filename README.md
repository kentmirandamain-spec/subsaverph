# SubSaverPH

SpaceX-inspired **text-only** storefront for discounted subscriptions.

## Products

- **Grok** (xAI) — Pro & Plus  
- **Canva** — Pro & Teams  
- **CapCut** — Pro & Standard+  
- **Netflix** — Premium & Standard  
- **YouTube** — Premium & Family  

## Features

- Black canvas UI (SpaceX reference) — **no images**
- **160+ payment currencies** (default PHP, live FX when online)
- Cart, checkout, demo redemption codes

## Run

```powershell
cd C:\Users\ADMIN\subsave
python -m http.server 8790
```

Open http://localhost:8790
